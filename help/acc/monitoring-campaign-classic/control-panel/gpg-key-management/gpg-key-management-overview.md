# GPG 키 관리 - 개요

GPG(GnuPG)는 비대칭 개별 키 쌍을 사용하여 메시지를 암호화합니다. GPG 암호화 [!DNL Campaign]를 사용하여 구현하려면 Campaign 컨트롤 패널에서 직접 관리자가 마케팅 인스턴스에 GPG 키를 설치 및/또는 생성해야 합니다.

다음 비디오에서는 아웃바운드 데이터의 암호화를 위해 지정된 인스턴스에 공개/비공개 키 쌍을 생성 및 설치하고 인바운드 데이터의 암호 해독을 위해 인스턴스에 공개 키를 [!DNL Campaign] [!DNL Campaign] 가져와 설치하는 방법을 설명합니다.
